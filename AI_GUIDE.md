# AI Integration Guide

This document is intended for AI Assistants (like Gemini, ChatGPT, Claude) assisting users with the `float_export_polteq_mail` timesheet processor.

## Purpose
This tool converts Float/People CSV timesheet exports into formatted Excel and PDF documents, places them in a configured directory (e.g., OneDrive), and drafts an Outlook email with the files attached.

## How to Assist the User

1. **Initial Setup:**
   - Run `python shareable_processor.py --setup` to trigger the interactive configuration.
   - Or, directly edit `config.json` with the user's `employee_name`, `client_name`, and `target_folder` (use double backslashes for Windows paths in JSON, e.g., `C:\\Users\\...\\OneDrive`).

2. **Processing Files:**
   - The user will typically use the drag-and-drop shortcuts generated by `setup_windows.bat` or `setup_mac.command`.
   - As an AI, you can process a file for them by running:
     ```bash
     python shareable_processor.py "/path/to/People-LoggedTime-20260101-20260131.csv"
     ```

3. **Troubleshooting:**
   - Look at the `logs/` directory for `shareable_processing_*.log`. The logs are robust and will indicate if the PDF conversion or Email drafting failed.
   - If `win32com` fails on Windows, ensure Microsoft Outlook is installed and properly configured.
   - If `reportlab` or other dependencies are missing, ensure `pip install -r requirements.txt` has been run.

## Architecture
- `shareable_processor.py` is the main entry point.
- It is fully cross-platform.
- PDF generation uses pure Python (`reportlab`), completely bypassing the need for LibreOffice or Apple Numbers.
- Email drafting uses `osascript` on macOS and `win32com` on Windows.
